Morse Code History

Morse code is a method of transmitting text information as a series of on-off tones, lights, or clicks that can be directly understood by a skilled listener or observer without special equipment...

Designer's Ideas

Morse code is a convenient way to send messages during WW2. This project can enable you to translate morse code into English quickly, and vice versa without reading too much reference. You can also translate text file into text file in the File to File section of tis program. Feel free to play with it!
